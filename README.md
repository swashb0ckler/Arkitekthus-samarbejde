# Arkitekthus-samarbejde

## Universitet projekt

Under studiet samarbejdede vi med et arkitekt firma. Meget arkitekt arbejde foregår er fysisk, hvor der bliver delt dokumenter fra hånd til hånd. 
Vores ide var at fjerne bottlenecks i arbejdet ved at lave en slags 'version-kontrol for arkitekter', hvor tegninger/sketches/planer nemt kan
findes frem og sendes. 

Virksomhedsnavnet er blevet fjernet fra filerne og logoet. 

Derudover kunne koden klart bruge et par refactors eller 3:)
